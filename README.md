# redux

This is my own implementation of the state management library [Redux](https://github.com/gaearon/redux).

## Installation

__NPM:__
```sh
npm install @chidumennamdi/redux -S
```
__Yarn:__
```sh
yarn add @chidumennamdi/redux
```

* createStore
* combineReducers
